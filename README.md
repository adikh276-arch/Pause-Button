# Pause Button

A simple application designed to help you take a pause and reflect.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- i18next (Internationalization)

## Development

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the development server:
   ```sh
   npm run dev
   ```

## Deployment

This project is configured to run at `/pause_button/`.

The project includes a Dockerfile and Nginx configuration for deployment behind a subpath.
